PROBLEMEN / KAN BETER:

1. Geen bevestiging voor verwijderen (KRITIEK PROBLEEM!)
   
   Invoer: belangrijkDocument.txt
   Probleem: Bestand wordt DIRECT verwijderd zonder "Weet je het zeker?" vraag
   Risico: Gebruiker kan per ongeluk belangrijke bestanden verwijderen
   Oplossing: Vraag "Weet je zeker dat je [bestand] wilt verwijderen? (ja/nee)"

2. Geen path validatie (KRITIEK SECURITY PROBLEEM!)
   
   Invoer: /System/Library/belangrijkBestand.dat
   Probleem: Kan systembestanden verwijderen als gebruiker rechten heeft
   Risico: Kan systeem onbruikbaar maken
   Oplossing: Beperk verwijderen tot een specifieke safe directory

3. Directory traversal mogelijk (SECURITY PROBLEEM!)
   
   Invoer: ../../../etc/passwd
   Probleem: Kan bestanden buiten de huidige directory verwijderen
   Risico: Toegang tot en verwijdering van bestanden waar je niet bij zou moeten kunnen
   Oplossing: Valideer dat het path geen ".." bevat

PROBLEMEN / KAN BETER:

1. Hardcoded credentials (GROOT SECURITY PROBLEEM)
   
   Probleem: Gebruikersnaam "admin" en wachtwoord "1234" staan rechtstreeks in de code
   Risico: Iedereen die de code ziet kent het wachtwoord
   Oplossing: Gebruik een database of configuratiebestand met gehashte wachtwoorden

2. Geen wachtwoord hashing (GROOT SECURITY PROBLEEM)
   
   Probleem: Wachtwoord staat als plaintext "1234" in de code
   Risico: Bij datalekken is het wachtwoord direct leesbaar
   Oplossing: Gebruik SHA-256 of bcrypt om wachtwoorden te hashen

3. Geen brute force protectie
   
   Invoer: 100x fout wachtwoord proberen
   Probleem: Programma blijft draaien, geen limiet op pogingen
   Oplossing: Blokkeer na 3 foute pogingen
